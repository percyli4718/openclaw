# OpenClaw 项目

## 项目说明
企业级 Agent 编排系统 - 基于 LangGraph + Harness/Context Engineering

## 技术栈
- **Python**: 3.11+
- **Agent 框架**: LangGraph
- **向量数据库**: Qdrant/Milvus
- **部署**: Docker + Kubernetes

## Superpowers + gstack 工作流程

| 阶段 | 技能 | 输出 |
|------|------|------|
| 阶段 0: 需求澄清 | /office-hours + brainstorming | Product Brief + Design Spec |
| 阶段 1: 计划与审查 | writing-plans + /autoplan | Implementation Plan |
| 阶段 2: 隔离环境 | using-git-worktrees | Git Worktree + Branch |
| 阶段 3: 编码实现 | subagent-driven-development + TDD | Source Code + Commits |
| 阶段 4: 调试验证 | systematic-debugging + verification | Fixed Code + Verification Report |
| 阶段 5: 质量门禁 | /qa + /cso + requesting-code-review | QA Report + Security Audit |
| 阶段 6: 发布 | /ship | GitHub Release |

## 技能分工

### Superpowers（思考与流程层）
- brainstorming, writing-plans, using-git-worktrees, subagent-driven-development, test-driven-development, systematic-debugging, verification-before-completion, requesting-code-review

### gstack（执行与外部世界层）
- /office-hours, /autoplan, /qa, /cso, /ship, /browse
